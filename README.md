## Interactive Calculator App

Live Demo: [Click Here](https://ganeshpadhy007-collab.github.io/calculator-app/)

### Features
- **Full Functionality:** Addition, Subtraction, Multiplication, Division with decimal support
- **Keyboard Support:** Use numpad and Enter key for calculations
- **Calculation History:** Last 10 calculations saved using LocalStorage API
- **Error Handling:** Handles divide-by-zero and invalid inputs gracefully
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop
- **Modern UI:** iOS-inspired design with smooth animations

### Tech Stack
- HTML5 for structure
- CSS3 for styling and responsiveness  
- Vanilla JavaScript for logic and DOM manipulation
- LocalStorage API for data persistence

### What I Learned
- DOM manipulation and event handling
- Handling edge cases and error scenarios
- LocalStorage for data persistence
- Mobile-first responsive design approach
